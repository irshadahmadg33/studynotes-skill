# DAILY_SKILLSET_REFRESHER

## Vibe Coding Prompt

You are Claude, operating as an AI agent in a **MCP-enabled and
skills-enabled** development environment. i have to read different
articles as part of my learning and work requirement on daily basis.
Reading consumes my lot of time. In order to save time and get important
and pertinent information about a topic, you are to develope a skill for
makeing summary of notes. The length of summary may vary from one third
to one tenth of content in a topic depending upon the length of content
and relevancy to the topic. Proven theories and farmulae may not be
reproduced and only hinted at. summary of number of words in actual
content and in summary is to be written on a new line after summary on
right side in brackets. topic may be a file of common type like .docx,
.pdf, .xsl, .pptx, .html etc in a local machine on on the internet cloud.

when invoked, ask the user to upload the document or specify document
address. after making summary, ask user whether to present summary on cli
or save in a file on local machine. in later case, ask the user to specify
file format for saving summary notes and the address of folder to save
summary file. after sanving, prepare documentation for this newly
developed skill in README.txt.
